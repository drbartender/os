# Dr. Bartender Placeholder Page

## What is included

- `placeholder-standalone.html` — a quick static page you can upload anywhere
- `DrBartenderPlaceholder.jsx` — React component version
- `dr-bartender-placeholder.css` — shared styling

## Quickest path live

Upload `placeholder-standalone.html` and `dr-bartender-placeholder.css` to any simple host.

## Drop into the React app

1. Copy `DrBartenderPlaceholder.jsx` into `client/src/pages/`
2. Copy `dr-bartender-placeholder.css` into `client/src/`
3. Import the CSS inside the component or in `client/src/index.css`
4. Add a route in `client/src/App.js`
5. Replace the email and phone placeholders

## Things to customize first

- Replace `hello@drbartender.com`
- Replace `+10000000000`
- Update nav links if your routes differ
- Swap in real art files once you have them
